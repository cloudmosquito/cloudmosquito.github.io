# 怎么解决 Ubuntu 关机后电脑主板不断电

## 问题描述
如题

## 解决方案

第一步：修改 grub 文件

```shell
sudo gedit /etc/default/grub

```

第二步：修改 GRUB_CMDLINE_LINUX_DEFAULT 这一行为

```shell
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_osi=! pci=noaer"
```

第三步：保存退出后更新 GRUB 配置

```shell
sudo update-grub
```

第四步：重启生效

```shell
sudo reboot
```
