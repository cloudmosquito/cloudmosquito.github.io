    дәҢж¬ЎеһӢдёҺпјҲеҚҠпјүжӯЈиҙҹе®ҡжҖ§

    1 ж ҮйҮҸеҮҪж•°зҡ„жӯЈ(иҙҹ)е®ҡжҖ§

    еҜ№дәҺдёҖдёӘnз»ҙз©әй—ҙпјҢеҸҜд»ҘеңЁдёҠе®ҡд№үдёҖдёӘж ҮйҮҸеҮҪж•°V(x)пјҢжҲ‘д»¬жҺҘдёӢжқҘи®Ёи®әиҝҷдёӘV(x)зҡ„жҖ§иҙЁгҖӮ

    • жӯЈе®ҡжҖ§пјҡ \(V(0) = 0, \forall x \neq 0, V(x) > 0\)
    • еҚҠжӯЈе®ҡжҖ§пјҡ\(V(0) = 0, \forall x \neq 0, V(x) \ge 0\)
    • иҙҹе®ҡжҖ§пјҡ\(V(0) = 0, \forall x \neq 0, V(x) < 0\)
    • еҚҠиҙҹе®ҡжҖ§пјҡ\(V(0) = 0, \forall x \neq 0, V(x) \le 0\)
    • дёҚе®ҡжҖ§пјҡ\(\exists x_1,x_2 \in \mathbb{R}^n, V(x_1) > 0, V(x_2) < 0\)

    2 дәҢж¬ЎеһӢзҡ„жӯЈ(иҙҹ)е®ҡжҖ§

    еҪ“ж ҮйҮҸеҮҪж•°V(x)жҳҜдәҢж¬ЎеһӢеҪўејҸпјҢд№ҹе°ұжҳҜиҜҙпјҢ\(V(x) = x^TAx\) пјҢе…¶дёӯ \(A\) жҳҜдёҖдёӘзәҝжҖ§е®ҡеёёзҹ©йҳөж—¶пјҢжҲ‘д»¬д№ҹдјҡиҜҙпјҢ\(A\) е…·жңүдёҺ \(V(x)\) дёҖиҮҙзҡ„ (еҚҠ)жӯЈ/иҙҹе®ҡжҖ§гҖӮ

    2.1 дәҢж¬ЎеһӢдёәд»Җд№ҲиҰҒжұӮзҹ©йҳөеҜ№з§°пјҹ

    иҝҷйҮҢжңүж•°еӯҰдёҠзҡ„дёҖдёӘTrickгҖӮ

    еҺ»зңӢеҗ„з§ҚеҸӮиҖғд№Ұдёӯе…ідәҺвҖңдәҢж¬ЎеһӢвҖқзҡ„е®ҡд№үпјҢе®ғд»¬дјҡе‘ҠиҜүдҪ пјҢдәҢж¬ЎеһӢй•ҝиҝҷдёӘж ·еӯҗ

    \[x^TAx = \sum\limits_{i}\sum\limits_{j}a_{i,j}x_ix_j\]

    дҪҶе®ғд»¬еҗҢж—¶д№ҹдјҡиҰҒжұӮ \(A\) жҳҜдёҖдёӘеҜ№з§°зҹ©йҳөпјҢиҝҷжҳҜдёәд»Җд№Ҳе‘ўпјҹ

    жҲ‘д»¬д»Һеӯ—йқўж„Ҹд№үдёҠжҖқиҖғвҖңдәҢж¬ЎеһӢвҖқиҝҷдёӘжҰӮеҝөпјҢеҫҲиҮӘз„¶дјҡи§үеҫ—пјҢеҰӮжһңж ҮйҮҸеҮҪж•° \(\sum\limits_{i}\sum\limits_{j}a_{i,j}x_ix_j\) еҸҜд»ҘиЎЁзӨәжҲҗе®Ңе…Ёе№іж–№еҪўејҸзҡ„зәҝжҖ§з»„еҗҲпјҢйӮЈд№Ҳе®ғжүҚз§°еҫ—дёҠжҳҜдёҖдёӘдәҢж¬ЎеһӢгҖӮ

    дҪҶе…¶е®һпјҢжүҖжңүж ҮйҮҸеҮҪж•° \(\sum\limits_{i}\sum\limits_{j}a_{i,j}x_ix_j\) йғҪиғҪиЎЁзӨәдёәе®Ңе…Ёе№іж–№еҪўејҸзҡ„зәҝжҖ§з»„еҗҲгҖӮ

    дёҫдёӘдҫӢеӯҗпјҢ\(x_1^2+2x_1x_2+2x_2^2 = (x_1+x_2)^2+x_2^2\) пјҢеҸӘиҰҒж №жҚ®иҖҰеҗҲйЎ№й…Қзі»ж•°пјҢеҶҚз•ҷеҮәеҚ•зӢ¬зҡ„е№іж–№йЎ№пјҢе°ұиғҪжҠҠд»»ж„ҸдёҖдёӘдәҢж¬ЎеһӢиҪ¬еҢ–дёәе®Ңе…Ёе№іж–№зҡ„еҪўејҸгҖӮ

    жүҖд»ҘпјҢеҜ№дәҺд»»ж„Ҹзҹ©йҳө \(A\) пјҢзҗҶи®әдёҠ \(x^TAx\) йғҪеә”иҜҘжҳҜз¬ҰеҗҲжҲ‘д»¬зӣҙи§үзҡ„дёҖдёӘдәҢж¬ЎеһӢгҖӮ

    йӮЈд№ҲеҸҜд»ҘеҗҲзҗҶжҖҖз–‘пјҢ\(\forall A, \exists B, B = B^T, x^TAx = x^TBx\) .

    дәӢе®һдёҠпјҢжҲ‘д»¬д№ҹзЎ®е®һеҸҜд»ҘжүҫеҲ°иҝҷж ·зҡ„дёҖдёӘ \(B\) пјҢ \(B = (A^T + A)/2\) .

    е…·дҪ“жҺЁеҜјиҝҮзЁӢеҰӮдёӢжүҖзӨә

    \[\begin{aligned}x^T(\frac{A^T+A}{2})x &= \frac{1}{2}(x^TA^Tx + x^TAx)\\ &= \frac{1}{2}[(x^TAx)^T+x^TAx]\\ &= \frac{1}{2}(x^TAx+x^TAx)\\ &=x^TAx\end{aligned}\]

    2.2 дәҢж¬ЎеһӢжӯЈе®ҡзҡ„е……иҰҒжқЎд»¶пјҡж–№йҳөзҡ„йЎәеәҸдё»еӯҗејҸйғҪеӨ§дәҺйӣ¶

    еЎһе°”з»ҙж–Ҝзү№еҮҶеҲҷпјҡдәҢж¬ЎеһӢжӯЈе®ҡзҡ„е……иҰҒжқЎд»¶жҳҜж–№йҳөзҡ„йЎәеәҸдё»еӯҗејҸйғҪеӨ§дәҺйӣ¶гҖӮ

    йЎәеәҸдё»еӯҗејҸпјҡеҜ№дәҺз»ҷе®ҡзҡ„ \(n\) йҳ¶ж–№йҳө \(A_n\) пјҢе…¶еүҚ \(k\) иЎҢ \(k\) еҲ—з»„жҲҗзҡ„ж–№йҳө \(A_{(k)}\) зҡ„иЎҢеҲ—ејҸ \(\Delta_k\) дёә \(A_n\) зҡ„ \(k\) йҳ¶йЎәеәҸдё»еӯҗејҸгҖӮ

    Sylvester Criterionпјҡ\(A_n \in \mathbb{R}^{n\times n}, A_n \text{жҳҜжӯЈе®ҡзҡ„, iff,}\ \Delta_1 > 0, \Delta_2 > 0, ..., \Delta_n > 0\) .

    2.2.1 иҜҒжҳҺеҮҶеӨҮвҖ”вҖ”еј•зҗҶ

    1. гҖҗеј•зҗҶ1гҖ‘пјҡжӯЈе®ҡзҹ©йҳөзҡ„зү№еҫҒеҖјеӨ§дәҺ0гҖӮ

      \(\forall x \neq 0, x^TAx = x^T\lambda_i x > 0\) пјҢе…¶дёӯ\(\lambda_i\) жҳҜAзҡ„з¬¬iдёӘзү№еҫҒеҖјгҖӮжүҖд»Ҙжңү \(\lambda_i > 0\) .

    2. гҖҗеј•зҗҶ2гҖ‘пјҡж–№йҳөзҡ„иЎҢеҲ—ејҸзӯүдәҺзү№еҫҒеҖјзҡ„з§ҜгҖӮ

      ж–№йҳөзҡ„зү№еҫҒеӨҡйЎ№ејҸ \(|A-\lambda E| = (\lambda_1-\lambda)(\lambda_2 - \lambda)...(\lambda_n -\lambda)\) гҖӮ

      пјҲжіЁж„ҸпјҡдёҚжҳҜ \((\lambda-\lambda_1)(\lambda-\lambda_2)...(\lambda-\lambda_n)\) пјҢеӣ дёәе·ҰејҸзҡ„иҙҹеҸ·еңЁ \(\lambda\) дёҠпјү

      еҸ– \(\lambda = 0\) пјҢе°ұеҫ—еҲ°дәҶ \(|A| = \prod_{i=1}^{n}\lambda_i\) .

    3. гҖҗеј•зҗҶ3гҖ‘пјҡжӯЈе®ҡзҹ©йҳөзҡ„иЎҢеҲ—ејҸеӨ§дәҺ0гҖӮ

      з”ұгҖҗеј•зҗҶ1гҖ‘гҖҗеј•зҗҶ2гҖ‘еҚіеҫ—гҖӮ

    4. гҖҗеј•зҗҶ4гҖ‘пјҡе®һеҜ№з§°зҹ©йҳөзҡ„дёҚеҗҢзү№еҫҒеҖјеҜ№еә”зҡ„зү№еҫҒеҗ‘йҮҸзӣёдә’жӯЈдәӨгҖӮ

      еҜ№дәҺдёҖдёӘе®һеҜ№з§°зҹ©йҳө \(A_n\) пјҢи®ҫе®ғзҡ„дёӨдёӘдёҚзӣёзӯүзҡ„зү№еҫҒеҖјдёә \(\lambda_1,\lambda_2\) пјҢеҲҶеҲ«еҜ№еә”зү№еҫҒеҗ‘йҮҸ \(x_1, x_2\) .

      жңү \(A_nx_1 = \lambda_1x_1\) пјҢе°ҶзӯүејҸдёӨиҫ№йғҪеҸ–иҪ¬зҪ®пјҢеҫ— \(x_1^TA_n^T = \lambda_1x_1^T\) пјҢеҶҚеҸід№ҳ \(x_2\) пјҢжңү

      \[x_1^TA_n^Tx_2 = \lambda_1x_1^Tx_2\]

      еҗҢзҗҶпјҢжңү \(A_nx_2 = \lambda_2x_2\) пјҢе·Ұд№ҳ \(x_1^T\) пјҢжңү

      \[x_1^TA_nx_2 = \lambda_2x_1^Tx_2\]

      еӣ дёә \(A_n^T=A_n\) пјҢжҲ‘д»¬еҸҜд»Ҙеҫ—еҲ° \((\lambda_1-\lambda_2)x_1^Tx_2 = 0\) пјҢеӣ дёә \(\lambda_1 \neq \lambda_2\) пјҢжүҖд»Ҙ\(x_1^Tx_2 = 0\) пјҢиҜҒжҜ•.

    2.2.2 иҜҒжҳҺз¬¬дёҖйғЁеҲҶвҖ”вҖ”е……еҲҶжҖ§

    е·ІзҹҘ \(A_n\) жӯЈе®ҡпјҢйңҖиҰҒиҜҒжҳҺ \(\Delta_k>0, 0<k\le n\) .

    • proofпјҡ

      еҜ№дәҺд»»ж„ҸдёҖдёӘ \(k\) з»ҙеҗ‘йҮҸ \(z_k = \{z_1,z_2,...,z_k\}^T\) пјҢ

      жҲ‘д»¬йғҪеҸҜд»Ҙжһ„йҖ дёҖдёӘ \(n\) з»ҙеҗ‘йҮҸ \(\mathbf{x}_{(k)} = \{x_1, x_2,...,x_k,0,0,...,0\}^T\) .

      еӣ дёә \(\mathbf{x_{(k)}}^TA\mathbf{x_{(k)}}>0\) пјҢжүҖд»Ҙ \(z_k^TA_{(k)}z_k>0\) жҒ’жҲҗз«ӢгҖӮ

      жүҖд»Ҙ \(A_{(k)}\) д№ҹжҳҜжӯЈе®ҡзҡ„пјҢж №жҚ®гҖҗеј•зҗҶ3гҖ‘пјҢ\(\Delta_k>0\) .

    2.2.3 иҜҒжҳҺз¬¬дәҢйғЁеҲҶвҖ”вҖ”еҝ…иҰҒжҖ§

    еҜ№дәҺз»ҷе®ҡж–№йҳө \(A_n\) пјҢе·ІзҹҘ \(\Delta_k>0, 0<k\le n\) пјҢйңҖиҰҒиҜҒжҳҺ \(A\) жӯЈе®ҡгҖӮ

    proofпјҡз”Ёж•°еӯҰеҪ’зәіжі•иҜҒжҳҺгҖӮ

    йҰ–е…ҲпјҢеҪ“n=1ж—¶пјҢ\(\Delta_1 = a_1 > 0\) пјҢжңү \(x^TA_{(1)}x = a_1x^Tx = a_1x^2 > 0, x \neq 0\) пјҢжүҖд»Ҙ \(A_{(1)}\) жҳҜжӯЈе®ҡзҡ„пјӣ

    е…¶ж¬ЎпјҢеҪ“n=2ж—¶пјҢ\(\Delta_1 = a_{11}>0, \Delta_2 = a_{11}a_{22}-a_{12}^2>0\) пјҢжңү

    \[x^TA_{(2)}x = a_{11}x_1^2+2a_{12}x_1x_2+a_{22}x_2^2 = a_{11}(x_1+\frac{a_{12}}{a_{11}}x_2)^2+\frac{a_{11}a_{22}-a_{12}^2}{a_{11}}x_2^2\]

    еҶҚиҖ…пјҢеҪ“ \(x_1,x_2\neq 0\) ж—¶пјҢ\(x^TA_{(2)}x\) жҒ’жҲҗз«ӢпјҢжүҖд»Ҙ \(A_{(2)}\) жҳҜжӯЈе®ҡзҡ„гҖӮ

    жҺҘдёӢжқҘйңҖиҰҒиҜҒжҳҺпјҢиӢҘ \(A_{(j)}\) жӯЈе®ҡдё” \(\Delta_{j+1}>0\) пјҢеҲҷ \(A_{(j+1)}\) д№ҹжӯЈе®ҡгҖӮе…¶дёӯ \(1\le j \le n-1\) .

    жҲ‘д»¬дҪҝз”ЁеҸҚиҜҒжі•гҖӮеҒҮи®ҫ \(A_{(j+1)}\) йқһжӯЈе®ҡпјҢеҲ©з”Ё \(\Delta_{j+1}>0\) еҜјеҮә \(A_{(j)}\) йқһжӯЈе®ҡгҖӮ

    ж №жҚ®гҖҗеј•зҗҶ1гҖ‘пјҢ\(A_{(j+1)}\) йқһжӯЈе®ҡпјҢеҲҷжңүдёҖдёӘе°ҸдәҺзӯүдәҺ0зҡ„зү№еҫҒеҖјгҖӮ

    еӣ дёә \(\Delta_{j+1}>0\) пјҢж №жҚ®гҖҗеј•зҗҶ2гҖ‘пјҢиҝҷдёӘзү№еҫҒеҖјеҝ…йЎ»е°ҸдәҺ0пјҢе№¶дё”еҗҢж—¶еӯҳеңЁеҒ¶ж•°дёӘиҙҹзү№еҫҒеҖјгҖӮ

    жҲ‘д»¬жҢ‘еҸ–е…¶дёӯ2дёӘиҙҹзү№еҫҒеҖј \(\lambda_a, \lambda_b\) пјҢе…¶еҜ№еә”зҡ„j+1з»ҙзү№еҫҒеҗ‘йҮҸеҲҶеҲ«дёә \(x_a,x_b\) гҖӮ

    йҖүжӢ© \(\alpha,\beta\) пјҢжҲ‘д»¬еҸҜд»Ҙеҫ—еҲ°дёҖдёӘж–°зҡ„j+1з»ҙеҗ‘йҮҸ \(u = \alpha x_a + \beta x_b \neq 0\) пјҢе…¶з¬¬j+1з»ҙеҖјдёә0.

    иҖғиҷ‘дәҢж¬ЎеһӢ

    \[\begin{aligned} u^TA_{(j+1)}u &= (\alpha x_a+\beta x_b)^TA_{(j+1)}(\alpha x_a+\beta x_b)\\ &= (\alpha x_a^T + \beta x_b^T)(\alpha A_{(j+1)}x_a + \beta A_{(j+1)}x_b)\\ &= (\alpha x_a^T + \beta x_b^T)(\alpha\lambda_ax_a+\beta\lambda_bx_b)\\ &= (\alpha^2\lambda_ax_a^Tx_a + \alpha\beta\lambda_bx_a^Tx_b + \beta\alpha\lambda_ax_b^Tx_a+\beta^2\lambda_bx_b^Tx_b)\\ &= (\alpha^2\lambda_ax_a^Tx_a + \beta^2\lambda_bx_b^Tx_b)\\ &= \alpha^2x_a^TA_{(j+1)}x_a + \beta^2x_b^TA_{j+1}x_b\\ & < 0\end{aligned}\]

    е…¶дёӯзҡ„еҢ–з®ҖиҝҮзЁӢеҲ©з”ЁдәҶгҖҗеј•зҗҶ4гҖ‘гҖӮ

    еӣ дёәuзҡ„з¬¬j+1з»ҙеҖјдёә0пјҢжүҖд»ҘеҲ©з”ЁеҲҶеқ—зҹ©йҳөзҡ„и®Ўз®—еҸҜзҹҘ \(u^TA_{(j+1)}u = u_{(j)}^TA_{(j)}u_{(j)} < 0\) пјҢиҝҷдёҺ \(A_{(j)}\) жӯЈе®ҡзӣёдә’зҹӣзӣҫпјҢеӣ жӯӨжҲ‘д»¬зҡ„еҒҮи®ҫй”ҷиҜҜпјҢ\(A_{(j+1)}\) жҳҜжӯЈе®ҡзҡ„гҖӮ

    з»јдёҠжүҖиҝ°пјҢжҲ‘д»¬е®ҢжҲҗдәҶ Sylvester Criterion зҡ„иҜҒжҳҺгҖӮ
