    е№іиЎЎзӮ№зҡ„зЁіе®ҡжҖ§

    иҜҘйғЁеҲҶеҶ…е®№дёҺ Lyapunov зЁіе®ҡжҖ§ жңүеӨ§йҮҸйҮҚеӨҚд№ӢеӨ„пјҢйҮҚеӨҚеҶ…е®№дёҚеҠ иөҳиҝ°гҖӮ

    иҖғиҷ‘дёҖдёӘ n з»ҙзҡ„иҮӘжІ»зі»з»ҹ \(\dot{\mathbf{x}} = f(\mathbf{x})\) пјҢе…¶е№іиЎЎзӮ№е®ҡд№үдёәж»Ўи¶і \(\dot{\mathbf{x}} = f(\mathbf{x}) = 0\) зҡ„зӮ№гҖӮ

    1 зәҝжҖ§зі»з»ҹзҡ„зЁіе®ҡжҖ§

    иҖғиҷ‘зәҝжҖ§зі»з»ҹ \(\dot{\mathbf{x}} = A\mathbf{x}\) пјҢеҰӮжһң \(A\) зҡ„жүҖжңүзү№еҫҒеҖј \(\lambda_i\) йғҪж»Ўи¶і \(Re\left[\lambda_i\right] < 0\) пјҢеҲҷз§°зҹ©йҳө \(A\) дёә Hurwitz зҹ©йҳөжҲ–иҖ…зЁіе®ҡзҹ©йҳөгҖӮ

    жҲ‘д»¬еҸҜд»ҘиҜҙжҳҺпјҢеҪ“дё”д»…еҪ“ \(A\) дёә Hurwitz зҹ©йҳөж—¶пјҢзәҝжҖ§зі»з»ҹеңЁеҺҹзӮ№еӨ„е…ЁеұҖжёҗиҝ‘зЁіе®ҡгҖӮ

    е®ҡзҗҶпјҡеҜ№дәҺд»»ж„Ҹ Hurwitz зҹ©йҳө A е’Ңд»»ж„ҸеҜ№з§°жӯЈе®ҡзҹ©йҳө Q пјҢйғҪеӯҳеңЁе”ҜдёҖзҡ„еҜ№з§°жӯЈе®ҡзҹ©йҳө P ж»Ўи¶і Lyapunov ж–№зЁӢпјҡ

    \[A^TP + PA = -Q\]

    иҜҒжҳҺпјҡ

    еҸ–

    \[P = \int_{0}^{+\infty}e^{A^\top t}Qe^{A t}\mathrm{d}t\]

    еӣ дёә A жҳҜ Hurwitz зҹ©йҳөпјҢжүҖд»Ҙзҹ©йҳөжҢҮж•° \(e^{At}\) д»ҘжҢҮж•°йҖҹеәҰиЎ°еҮҸпјҢз§ҜеҲҶеҖјж”¶ж•ӣгҖӮжӯӨж—¶пјҢжңү

    \[\begin{aligned} A^TP+PA &= \int_{0}^{+\infty}A^\top e^{A^\top t}Qe^{A t}\mathrm{d}t + \int_{0}^{+\infty}e^{A^\top t}Qe^{A t}A\mathrm{d}t\\ &= \int_{0}^{+\infty}\frac{\mathrm{d}}{\mathrm{d}t}\left(e^{A^\top t}Qe^{A t}\right)\mathrm{d}t\\ &= \left(e^{A^\top t}Qe^{A t} \right)|_{0}^{+\infty}\\ &= -Q \end{aligned}\]

    P зҡ„еҜ№з§°жҖ§жҳҫиҖҢжҳ“и§ҒгҖӮжҲ‘д»¬жҺҘдёӢжқҘиҜҒжҳҺ P зҡ„жӯЈе®ҡжҖ§е’Ңе”ҜдёҖжҖ§гҖӮ

    \[\begin{aligned} x^\top P x &= \int_{0}^{+\infty}x^\top e^{A^\top t}Qe^{A t}x\mathrm{d}t\\ &= \int_{0}^{+\infty}\left(e^{A t}x\right)^\top Qe^{A t}x\mathrm{d}t > 0 \end{aligned}\]

    дёҠејҸжҲҗз«ӢжҳҜеӣ дёә Q зҡ„жӯЈе®ҡжҖ§гҖӮ

    еҒҮи®ҫжңүеҸҰдёҖдёӘдёҚеҗҢдәҺ \(P\) зҡ„еҜ№з§°жӯЈе®ҡзҹ©йҳө \(\tilde P\) ж»Ўи¶іиҰҒжұӮпјҢеҲҷжңү

    \[A^\top \left(\tilde P-P\right) +\left(\tilde P-P\right) A = 0\]

    еҜ№дәҺйқһйӣ¶еҗ‘йҮҸ \(x\) пјҢеј•е…ҘдёҖдёӘиҫ…еҠ©еҮҪж•° \(\phi(x,t) = \left(e^{A t}x\right)^\top\left(\tilde P-P\right)\left(e^{A t}x\right)\)пјҢе…¶еҜ№ж—¶й—ҙзҡ„дёҖйҳ¶еҜјж•°дёә

    \[\begin{aligned} \dot{\phi} &= \left(A e^{A t}x\right)^\top \left(\tilde P - P\right)\left(e^{At}x\right) + \left(e^{A t}x\right)^\top \left(\tilde P - P\right)\left(Ae^{At}x\right)\\ &= \left(e^{A t}x\right)^\top \left[A^\top \left(\tilde P-P\right) +\left(\tilde P-P\right) A\right] \left(Ae^{At}x\right)\\ &= 0 \end{aligned}\]

    жүҖд»ҘеҮҪж•° \(\phi(x,t)\) е…ідәҺж—¶й—ҙжҳҜдёӘеёёеҖјеҮҪж•°гҖӮеӣ дёә A жҳҜ Hurwitz зҹ©йҳөпјҢжүҖд»Ҙ \(\lim_{t\to \infty} \phi = 0\) пјҢжүҖд»Ҙ \(\phi \equiv 0\) пјҢжүҖд»Ҙ \(\tilde P = P\) гҖӮ

    2 жҢҮж•°зЁіе®ҡжҖ§

    жҲ‘д»¬еҸӘиҖғиҷ‘зі»з»ҹе№іиЎЎзӮ№еңЁеҺҹзӮ№зҡ„жғ…еҶөгҖӮе№іиЎЎзӮ№дёҚеңЁеҺҹзӮ№зҡ„жғ…еҶөдёҺд№Ӣзӯүд»·гҖӮ

    2.1 е®ҡд№ү

    иҖғиҷ‘дёҖдёӘйқһзәҝжҖ§зі»з»ҹ \(\dot{\mathbf{x}} = f(\mathbf{x})\) пјҢе…¶дёӯ \(f(\mathbf{x})\) жҳҜдёҖдёӘе®ҡд№үеҹҹеҢ…еҗ«еҺҹзӮ№зҡ„еұҖйғЁ Lipschitz еҮҪж•°пјҢдё” \(f(0) = 0\) пјҢеҰӮжһңеӯҳеңЁжӯЈеёёж•° \(c, k, \lambda\) дҪҝеҫ—

    \[\left\|\mathbf{x}(t)\right\| \le k\left\|\mathbf{x}(0)\right\|e^{-\lambda t}, \forall t \ge 0\]

    еҜ№дәҺжүҖжңүеҺҹзӮ№йӮ»еҹҹеҶ…зҡ„еҲқеҖј \(\left\|\mathbf{x}(0)\right\| < c\) йғҪжҲҗз«ӢпјҢеҲҷз§°зі»з»ҹзҡ„е№іиЎЎзӮ№ \(\mathbf{x} = 0\) жҳҜжҢҮж•°зЁіе®ҡзҡ„гҖӮеҰӮжһң \(c\to \infty\) пјҢеҲҷз§°еҺҹзӮ№жҳҜе…ЁеұҖжҢҮж•°зЁіе®ҡзҡ„гҖӮ

    2.2 еҲӨе®ҡе®ҡзҗҶ

    жҲ‘д»¬иҰҒжұӮ \(f\) еңЁеҺҹзӮ№еӨ„жҳҜеҸҜеҫ®зҡ„гҖӮ

    еҰӮжһң \(f(\mathbf{x})\) зҡ„йӣ…еҸҜжҜ”зҹ©йҳө \(\dfrac{\partial f}{\partial \mathbf{x}}\) еңЁеҺҹзӮ№еӨ„зҡ„жүҖжңүзү№еҫҒеҖје®һйғЁеқҮе°ҸдәҺ 0 пјҲеҚі \(\dfrac{\partial f}{\partial \mathbf{x}}|_{\mathbf{x} = 0}\) жҳҜ Hurwitz зҹ©йҳөпјүпјҢеҲҷиҜҘзі»з»ҹеңЁеҺҹзӮ№еӨ„жҳҜжҢҮж•°зЁіе®ҡзҡ„гҖӮ

    2.3 жҖ§иҙЁ

    жҢҮж•°зЁіе®ҡжҖ§жҳҜдёӘжҜ”жёҗиҝ‘зЁіе®ҡжҖ§жӣҙејәзҡ„жҖ§иҙЁгҖӮиҝҷйҮҢвҖңејәвҖқзҡ„ж„ҸжҖқжҳҜпјҢжҢҮж•°зЁіе®ҡжҖ§жҳҜжёҗиҝ‘зЁіе®ҡжҖ§зҡ„е……еҲҶжқЎд»¶гҖӮиҝҷеҫҲе®№жҳ“иҜҒжҳҺпјҢ \(kc\) зӣёеҪ“дәҺжёҗиҝ‘зЁіе®ҡжҖ§йҮҢзҡ„ \(\varepsilon\) пјҢ \(c\) зӣёеҪ“дәҺжёҗиҝ‘зЁіе®ҡжҖ§йҮҢзҡ„ \(\delta\) пјҢеҜ№дәҺеҲқеҖј \(\left\|\mathbf{x}(0)\right\| < c = \delta\) зҡ„жғ…еҶөпјҢзі»з»ҹзҠ¶жҖҒ \(\left\|\mathbf{x}(t)\right\|\) е§Ӣз»ҲдёҚдјҡи¶…иҝҮ \(kc=\varepsilon\) пјҢе№¶дё”еҪ“ \(t\to \infty\) зҡ„ж—¶еҖҷпјҢ \(\left\|\mathbf{x}(t)\right\| \to 0\) гҖӮ
